### 3. Модуль пользовательского интерфейса (UI)
**Назначение:** Обеспечивает взаимодействие с пользователем через командную строку (CLI).

```markdown
# CONTRACT.md — UI Module

## Назначение
Интерпретирует команды пользователя и отображает данные, делегируя выполнение модулю Logic.

## Интерфейс

### `run()`
Запускает основной цикл приложения (ожидание ввода команд).
- **Параметры:** нет
- **Возвращает:** нет (бесконечный цикл до команды `exit`)

### `displayTasks(tasks)`
Форматирует и выводит список задач в консоль.
- **Параметры:** `tasks` (array) — массив объектов Task.
- **Возвращает:** `void`

### `showHelp()`
Выводит список доступных команд.
- **Параметры:** нет
- **Возвращает:** `void`

## Команды пользователя (Пример)
- `add <title>` — вызывает `logic.createTask`
- `done <id>` — вызывает `logic.completeTask`
- `list` — вызывает `logic.getPendingTasks` и передает результат в `displayTasks`

## Пример использования
```bash
$ task-cli add "Fix bug"
> Task created with ID 101

$ task-cli list
> 1. [ ] Fix bug
